## AcckCloud 是什么来头？

AcckCloud，域名 acck.io，2024 年下半年正式开业，是一家面向华人用户的小众 VPS 服务商。

技术架构用的是 KVM 虚拟化，这个没什么好说的，主流选择。目前覆盖四个地区的数据中心：

- **香港**（HKLite、HKVV 两个节点）
- **日本**（JPLite、JPIIJ 两个节点）
- **台湾**（TWLite 节点）
- **美国洛杉矶**（NUS9929 节点，2026 年 2 月新上线）

套餐分两类：流量型云服务器（按月流量计费，超出限速）和独享型云服务器。支持 Debian、Ubuntu、RockyLinux 等常见 Linux 系统，支付方式支持支付宝、USDT、账户余额，对国内用户很友好。

最大的卖点是内置了 ACCK DNS，香港节点解锁香港流媒体、日本节点解锁日本流媒体，支持 Netflix、Disney+、YouTube、ChatGPT、Abema、DMM、MytvSuper 等平台。

## 适合哪类用户？

坦白说，AcckCloud 的定位很清晰——就是给预算有限、需要流媒体解锁落地机的用户准备的。

**适合的场景：**

- 想看 Netflix、Disney+ 日区/港区内容，需要一台低价落地机
- 跑一些不需要实时交互的小项目，比如爬虫、挂机器人、反代等
- 低成本测试日本或香港节点的网络质量
- 想用美国洛杉矶原生 IP，预算不多的情况下 9929+CMIN2 双优化线路性价比很高

**不太适合的场景：**

- 需要稳定回国直连优化的业务（除了新出的美国 LAX9929 节点之外，香港和日本的套餐走国际线路，晚高峰可能有延迟波动）

## 几个需要知道的细节

**流量达量后怎么办？** 超出月流量上限后限速至共享 10Mbps，基本只能做些轻量操作。想恢复正常速度需要付费重置，费用因套餐不同，约 ¥5–¥24.88/次不等。如果流量消耗量大，建议上 Mini 或 Plus 套餐。

**支付方式**：支持支付宝、USDT、账户余额，国内用户充值购买没有门槛。

**操作系统**：支持 Debian、Ubuntu、RockyLinux 等主流 Linux 发行版，纯 Linux 环境，没有 Windows 选项。

**IP 配置**：大部分套餐默认提供 1 个 IPv4 + 1 个 IPv6（部分套餐仅有 IPv4，下单前注意查看套餐说明）。
